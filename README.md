# Blood Bank Management System

## Overview
The **Blood Bank Management System** is a Java-based desktop application designed to efficiently manage blood donation and distribution records in hospitals or blood banks. The system maintains structured records of blood donors and receivers, enabling quick lookup, updates, and tracking of blood availability by blood group.

This project focuses on reducing manual record-keeping, minimizing errors, and improving operational efficiency in blood bank management.

---

## Key Features
- Manage donor records (add, update, delete)
- Manage receiver records
- Track blood groups and availability
- Worker and admin login functionality
- Secure database-backed data storage
- User-friendly graphical interface

---

## Technologies Used
- **Programming Language:** Java  
- **UI Framework:** Java Swing (JSwing)  
- **Database:** MySQL  
- **IDE:** Eclipse  
- **JDBC Driver:** MySQL Connector/J  

---

## System Requirements
- Java JDK 8 or above  
- MySQL Server  
- Eclipse IDE (recommended)  
- MySQL JDBC Connector  

---

## Project Structure
- Java source files handle business logic and UI
- MySQL database stores donor, receiver, and transaction data
- Swing-based GUI for interaction
